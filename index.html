SWGA

Workflow to select primers for selective whole-genome amplification (SWGA)

Download this project as a .zip file Download this project as a tar.gz file

Welcome to SWGA.

Selective whole-genome amplification (SWGA) is a way to specifically amplify a genome above a background. This code is a self-contained workflow for identifying primer sets for use in this process.

Quick start:

Follow the installation instructions here, then simply follow these instructions in a directory of your choice:

./activate_swga
swga init
swga autopilot

All the parameters can be fine-tuned in a configuration file called default_parameters.cfg in the directory you ran swga init. For tips and tricks for optimizing these parameters, see the SWGA wiki.
